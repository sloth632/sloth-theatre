## Features

### Streaming
- Movies, TV shows, and Anime support
- Fast iframe player
- Server switch (Server 1 / Server 2)
- Episode autoplay and next episode system

### Discovery
- Trending page for movies & TV shows
- Search system
- Recommendations based on selected content

### User
- Full info page for each content
- Cover image, title, and overview
- Related recommendations displayed
- Back button for easy navigation
